# AWS Certified Security - Specialty (SCS-C03)

## Exam Information

| Aspect | Details |
|--------|---------|
| **Exam Code** | SCS-C03 |
| **Level** | Specialty |
| **Duration** | 170 minutes |
| **Questions** | 65 questions |
| **Passing Score** | 750/1000 |
| **Format** | Multiple choice, multiple response |
| **Cost** | $300 USD |
| **Prerequisite** | 5+ years IT security experience recommended |

## Exam Tips Summary

1. **Know GuardDuty finding types** and remediation actions
2. **Understand Security Hub standards** (CIS, AWS Foundational)
3. **Master KMS key policies** and grants
4. **Know IAM policy evaluation** logic
5. **Understand VPC security** (SG, NACL, Flow Logs)
6. **Practice incident response** scenarios
7. **Know encryption options** for each service
8. **Understand Organizations** and SCPs
9. **Know CloudTrail** log analysis
10. **Master cross-account access** patterns
